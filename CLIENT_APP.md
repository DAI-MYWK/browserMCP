# MCPクライアントアプリの作成

はい、MCPクライアントアプリも作成できます！

## MCPクライアントとは

MCPクライアントは、MCPサーバーに接続してツールを呼び出すアプリケーションです。

### 現在の構成

```
Claude Desktop (MCPクライアント)
    ↓
Browser MCP Server (MCPサーバー) ← 現在のプロジェクト
    ↓
Playwright (ブラウザ操作)
```

### カスタムクライアントの例

以下のようなクライアントアプリを作成できます：

1. **コマンドラインクライアント** - ターミナルからブラウザ操作
2. **Webアプリ** - ブラウザ上で動作するUI
3. **デスクトップアプリ** - Electronなどを使ったGUIアプリ
4. **VS Code拡張** - VS Code内でブラウザ操作
