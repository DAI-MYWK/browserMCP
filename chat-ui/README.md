# Browser MCP Chat UI

ブラウザ操作のためのチャットUIアプリケーションです。**GPTを使って自然言語の指示をマルチターンで処理**します。

## セットアップ

```bash
cd chat-ui
npm install
```

## OpenAI APIキーの設定

`.env.local`ファイルを作成し、OpenAI APIキーを設定してください：

```bash
cp .env.local.example .env.local
# .env.localを編集してOPENAI_API_KEYを設定
```

`.env.local`の内容：

```env
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_MODEL=gpt-5
```

**推奨モデル**:
- `gpt-5` - **最新モデル（推奨）** - 最高性能、マルチモーダル対応（temperatureはデフォルトの1固定）
- `gpt-4o` - 高精度だがGPT-5より旧世代
- `gpt-4o-mini` - コスト効率重視（temperature指定不可）
- `gpt-4-turbo` - バランス型

## 起動方法

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

## 使い方

1. **環境変数を設定**: `.env.local`に`OPENAI_API_KEY`を設定
2. **サーバーに接続**: 画面右上の「サーバーに接続」ボタンをクリック
3. **指示を入力**: チャット欄に自然言語でブラウザ操作の指示を入力
4. **実行**: 「送信」ボタンをクリック

## 使用例

### 複雑なタスクを一度に指示

```
Googleフォームで新しいアンケートを作成して、
タイトルに「顧客満足度調査」と入力して、
ラジオボタンの質問を追加して、
選択肢を「満足」「普通」「不満」にして
```

GPTが自動的に以下を実行します：
1. Googleフォームにアクセス
2. フォームタイトルを入力
3. 質問を追加
4. 質問タイプを選択
5. 選択肢を設定

### 複数ステップの操作

```
Airレジの予約ページを開いて、
予約日時を選択して、
フォームに以下の情報を入力してください：
- 名前: 田中太郎
- フリガナ: タナカタロウ
- メール: taro@example.com
- 電話: 0312345678
```

GPTが順次実行します：
1. URLに移動
2. 要素をクリック
3. 日時を選択
4. 各フィールドに入力

### スクリーンショットで確認しながら進める

```
Googleにアクセスして、
検索ボックスに「Playwright」と入力して、
検索ボタンをクリックして、
結果ページのスクリーンショットを取得して
```

各ステップの結果を確認しながら進めます。
